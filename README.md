# 🎲 骰子比大小 - 多人在线游戏

一个可以和朋友在线玩的骰子游戏，支持多人实时对战。

## 功能特性

- 🎯 **房间系统**: 创建或加入房间，与朋友一起玩
- 🎲 **骰子机制**: 3个骰子，点数总和 3-18
- 🎰 **轮流庄家**: 每轮自动轮换庄家
- 💰 **虚拟积分**: 每人初始 1000 积分
- 📊 **倍率系统**: 庄家可选择比大或比小，根据点数计算倍率
  - 比大: 高点数赢，倍率 0.5x - 3x
  - 比小: 低点数赢，倍率 0.5x - 3x
- 🎨 **精美界面**: 现代化 UI 设计

## 游戏规则

1. **准备阶段**: 所有玩家点击"准备好了"开始游戏
2. **下注阶段**: 每个玩家下注（10-当前积分）
3. **骰子阶段**: 系统自动骰3个骰子
4. **庄家选择**: 庄家可以看到骰子结果，选择"比大"或"比小"
5. **揭晓结果**: 根据庄家选择和点数计算输赢
6. **下一轮**: 庄家轮换，继续游戏

## 倍率计算

### 庄家选择比大
- 点数 ≥ 16: 3x 倍率
- 点数 ≥ 14: 2.5x 倍率
- 点数 ≥ 12: 2x 倍率
- 点数 ≥ 11: 1.5x 倍率
- 点数 ≤ 10: 0.5x 倍率（不利选择）

### 庄家选择比小
- 点数 ≤ 5: 3x 倍率
- 点数 ≤ 7: 2.5x 倍率
- 点数 ≤ 9: 2x 倍率
- 点数 ≤ 10: 1.5x 倍率
- 点数 ≥ 11: 0.5x 倍率（不利选择）

## 快速开始

### 前提条件
- Node.js 16+ 
- npm 或 yarn

### 安装和运行

#### 1. 启动后端服务器

```bash
cd server
npm install
npm run dev
```

后端将运行在 `http://localhost:3001`

#### 2. 启动前端界面（新终端）

```bash
cd client
npm install
npm run dev
```

前端将运行在 `http://localhost:3000`

### 开始游戏

1. 打开浏览器访问 `http://localhost:3000`
2. 输入你的名字
3. 点击"创建房间"或输入房间号"加入房间"
4. 让你的朋友在另一个浏览器窗口输入房间号加入
5. 所有人都点击"准备好了"开始游戏
6. 下注、享受游戏！

## 技术栈

### 后端
- **Node.js** + **Express**: HTTP 服务器
- **Socket.IO**: 实时双向通信
- **CORS**: 跨域支持

### 前端
- **React 18**: UI 框架
- **TypeScript**: 类型安全
- **Vite**: 快速构建工具
- **Socket.IO Client**: 实时通信客户端
- **CSS3**: 动画和样式
